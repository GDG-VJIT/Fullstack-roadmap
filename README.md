# 🚀 Full-Stack Web Development Roadmap (GDG CVJIT)

## ⚠️ Before you start…

Using AI tools is completely fine (and honestly, smart).

But don’t just copy-paste.

👉 Try to understand what’s happening  
👉 Try to build things on your own  
👉 Break stuff. Fix stuff. That’s how you grow  

AI should assist you — not replace you.

---

## 🌱 The Journey Begins

Every developer starts confused.

You see big projects… fancy apps… AI tools…  
And you think: “Where do I even start?”

This roadmap is your answer.

Not just steps —  
A path you walk, project by project, until things start making sense.

## ⚠️ The Developer Rule 😅

> If your API key is on GitHub… it’s no longer your API key.

Always:
- Use `.env`  
- Add `.env` to `.gitignore`
